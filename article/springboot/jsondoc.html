# jsondoc

为了让API使用者更好的调用，仅仅使API变得优秀是不够。API调用者还需要理解API的作用，并且知道如何去调用它。

同时，我们经常需要测试Rest API，将测试程序直接植入到应用程序将使工作变得更加轻松。

# JSONDoc

JSONDoc是一个用于构建RESTFul services文档的Java库。

  • 你可以在任何MVC框架中使用JSONDoc。
  • 它有一个有用的UI模块来显示生成的文档，还包括一个Playground来测试API。
  • 它是超轻量级的，少于250 Kb!

# 添加依赖

pom.xml

<dependency>
    ...
    <groupId>org.jsondoc</groupId>
        <artifactId>spring-boot-starter-jsondoc</artifactId>
        <version>1.2.19</version>
    </dependency>
    
    <dependency>
        <groupId>org.jsondoc</groupId>
        <artifactId>jsondoc-ui-webjar</artifactId>
    <version>1.2.19</version>
    ... 
</dependency>

# 配置

src/main/resources/application.properties

#jsondoc
jsondoc.version=1.0
jsondoc.basePath=http://localhost:8080
jsondoc.packages[0]=com.example.demo.controller #配置JSONDoc扫描的包
jsondoc.packages[1]=com.example.demo.api        #配置多个包

# 启用JSONDoc

在主配置类上加上@EnableJSONDoc：

src/main/java/com/example/demo/DemoApplication.java

@SpringBootApplication
@EnableJSONDoc
public class DemoApplication {

	public static void main(String[] args) {
		SpringApplication.run(DemoApplication.class, args);
	}
}

# 添加API相关的文档信息

在这里只简单使用了@Api, @ApiMethod。更多相关信息请查看JSONDoc Annotations。

src/main/java/com/example/demo/controller/AdminController.java

@Api(name = "Admin", description = "Api for managing admin.")
@RestController
public class AdminController {
    
    public @Autowired AdminService adminService;
    
    @RequestMapping("/admins")
    public List<Admin> getAllAdmins() {
        return adminService.findAllAdmins();
    }
    
    @ApiMethod(description = "Update admin and return admin.")
    @RequestMapping(value = "/admin", method = RequestMethod.PUT)
    public Admin updateAdmins(@RequestBody Admin admin) {
    	return adminService.updateAdmin(admin);
    }
}

# 查看文档和测试API

访问http://localhost:8080/jsondoc-ui.html, 然后在输入框中写入http://localhost:8080/jsondoc, 点击Get documentation后，你就可以看到文档了。

如上图所示， 在Body Object中输入Admin的JSON对象，点击Submit，就可以测试Update Admin的API。

如果你没有添加任何JSONDoc相关的注解在配置的包（com.example.demo.controller，com.example.demo.api ）的类里面，JSONDoc会为包里面相关的类(不是所有的类)默认生成一些信息。
